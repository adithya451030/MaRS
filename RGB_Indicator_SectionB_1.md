This project can be applied at Parking System of the vehicals, Small Robotics or Rover projects to avoid obstacles, Industries and many more.
This project is about how an ultrasonic sensor and a potentiometer controls a RGB LED. when the sensor readings is too far then the RGB glows with green colour, when the readings is near then the RGB glows with Red colour and if it is too near then the RGB glows with Red colour and it blinks faster, and when the reading is moderate then RGB glows with yellow colour.
At the stage of building circuit first I have taken arduino connected to the breadboard. After that A RGB LED is fixed to a random place of breadboard and connected its cathode to ground and other pins are connected to digital PWM pins of arduino board along with a 220 Ohm resistor to each pin. After that I have taken a potentiometer on the breadboard and connected it VCC, Wiper and GND to 5V, analog pin (A0), and ground respectivly. After then an Ultrasonic Distance sensor is taken an connected its VCC, Trig, Echo and GND to 5V, digital PWM pin of arduino, normal pin of arduino and ground respectively.
In this program time stores how long the echo signal takes to return, and distance stores the calculated distance in centimeters. previousMillis stores the last time the LED blinking was updated, so the code can blink without using delay(). ledState stores whether the blinking LED is currently ON or OFF. sensorValue stores the potentiometer reading. In loop(), the code first sends a short pulse from the trigger pin. The echo pin receives the reflected signal, and pulseIn() measures the return time. Using this time, the program calculates the distance. Then the potentiometer values some change into smaller range. So the converted potentiometer values is called threshold. Now this threshold decides how the LED should respond by comparing with the distance. The code also reads the potentiometer again into sensorValue, converts it into val, and uses that as the LED brightness level.
Coming to the logic if he distance is greater than the threshold, the object is far, so the green LED turns ON. If the distance is smaller than the threshold but still greater than half of it, the object is at medium range, so red and green turn ON together to make yellow. If the distance is less than that but still above 5 cm, the object is near, so only red turns ON. If the distance is 5 cm or below, the object is very close, so the red LED blinks. The blinking is done with help of millis function.
