At the stage of building the circiut, I have taken a LED and connected its cathode to ground and anode to arduino pin along with resistor of 220 Ohms. And then a push button is taken and connected its sideways adjcent pins (1b,2b) to arduino pin and ground.
In this millis function is used and randomDelayTime holds a random waiting time between 2 to 5 seconds. Specifically 2 to 5 sec is the best setting that avoid fastest and slowest.
In state 0, if the user presses the button during this waiting period, it detects it as an early press, prints a warning message, and resets the timer with a new random delay. In state 1, when the button is pressed, a small delay is used for debouncing, and then the reaction time is calculated as the difference between the current time and startTime. This value is printed to the Serial Monitor, and the LED is turned OFF.In state 2, the program pauses for 2 seconds before starting the next round. After the pause, a new random delay is generated, and the program returns to state 0.
