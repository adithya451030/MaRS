Initially all three LED's cathode is grounded to arduino and anode is connected to 220 Ohm resistor(for better brightness) and to the  pins of the arduino. 
In the code, referring the  millis() I have taken two variables called currentMillis which tells about current time and previouMillis which tells records the time when the last time an even happens. another variables called interval which denoted the time of LED should ON, timedelay which is used to keep the LED OFF.
At the beginning of code, It first checks whether the required time is passed or not. The program checks if the required interval has passed, and if so, it moves to the next step using a state variable. In each state, the LEDs are either turned ON or OFF. The ON time is always fixed at 1 second, while the OFF time increases gradually using the timedelay variable, starting from 500 ms and increasing by 500 ms each cycle. 
At state 1, LED turns ON an waits for 1 sec. At state 2, LED turns OFF for 500 ms and also time timedelay changes to 1000. At state 3, again LED turns ON for 1 sec. At state 4, LED turns OFF for updated time that means 1000 ms and again time delay increases by 500 ms. At stage 5, again LED turns ON for 1 sec. At last stage, the LED turns OFF for 1500 ms and then it directs to state 1. This goes infinite times until we stop.
